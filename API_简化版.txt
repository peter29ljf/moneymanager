# 资产管理系统 API 简化使用说明



## API 命令示例

### 获取所有资产
curl http://localhost:3000/api/assets

### 添加加密货币
curl -X POST http://localhost:3000/api/assets/crypto -H "Content-Type: application/json" -d '{"name":"比特币","price":68400,"quantity":0.5}'

### 添加股票
curl -X POST http://localhost:3000/api/assets/stocks -H "Content-Type: application/json" -d '{"name":"苹果公司","price":150,"quantity":100}'

### 更新加密货币价格
curl -X PUT http://localhost:3000/api/assets/crypto/ID -H "Content-Type: application/json" -d '{"price":69000}'

### 更新股票价格
curl -X PUT http://localhost:3000/api/assets/stocks/ID -H "Content-Type: application/json" -d '{"price":160}'

### 更新加密货币持仓数量
curl -X PUT http://localhost:3000/api/assets/crypto/ID -H "Content-Type: application/json" -d '{"quantity":1.2}'

### 更新股票持仓数量
curl -X PUT http://localhost:3000/api/assets/stocks/ID -H "Content-Type: application/json" -d '{"quantity":200}'
